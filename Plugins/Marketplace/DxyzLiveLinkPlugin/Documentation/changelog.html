Changelog¶

1.2.1¶

  • Release date: 05/2020

  • News :

    • Add 4.25 support

1.2.0¶

  • Release date: 04/2020

  • News :

    • Add a feedback GUI widget for getting Grabber preview

    • Add two calibration GUI widgets to customize the animation at runtime

    • Add support for MacOS distributions

  • Fixes :

    • Fix wrong timecode format in the Grabber remote controller

1.1.3¶

  • Release date: 12/2019

  • News :

    • Add 4.24 support

1.1.2¶

  • Release date: 08/2019

  • News :

    • Minor optimizations in reception loop

    • Add 4.23 support

1.1.1¶

  • Release date: 04/2019

  • News :

    • Add the reception FPS in the Live Link metadata

    • Add the animation computation framerate

    • Interface the DxyzLiveLink timecode with the LiveLink timecode (UE4.21 and above)

1.1.0¶

  • Release date: 03/2019

  • News :

    • Read the retargeting profile name from the ESC

    • Allow the user to override Live Link Subject name

    • Allow the user to clamp translations, rotations, or blendshapes values in the Retarget Asset

    • Identify the potential wrong shapes and bones in the Retarget Asset

    • Use “dxyz” extension for config files to avoid continuous file modification detection warning

    • Add a remote trigger for Grabber recording

    • Allow the user to hide the connection attempts logs

    • Add 4.22 support

1.0.3¶

  • Release date: 12/2018

  • Fixes :

    • Fix ESC parsing

    • Normalize rotations to avoid random assertion failure

1.0.2¶

  • Release date: 11/2018

  • News :

    • Expose the frame timecode to the Live Link component

    • Change the automatic mapping method

1.0.1¶

  • Release date: 10/2018

  • Fixes :

    • Fix packaging null reference exception in UE 4.20

1.0.0¶

  • Release date: 08/2018

  • News :

    • Remove DxyzLiveLinkInstance class

    • Allow blendshapes remapping in the RetargetAsset

    • Add documentation in the editor

  • Fixes :

    • Fix translations percentages in the RetargetAsset

    • Fix bones and curves display in UE_4.20.2

0.0.2¶

  • Release date: 06/2018

  • News :

    • Add a warning when a source contains non-unique blendshape names

    • Add a warning when a rig contains non-zero rotation values
